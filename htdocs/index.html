CSLiS Documentation

CSLiS is a software package that is based on the Linux STREAMS (LiS) open source package, LiS-2.19 level. It comprises an implementation of SVR4 compatible STREAMS for Linux. It takes the form of a loadable module for the Linux kernel. CSLiS should be install in the source directory, /usr/src/CSLiS-7xx on your system, not in the kernel source tree. When it is built it is possible to link pre-compiled STREAMS drivers with it so that when CSLiS loads into the kernel it brings "application" drivers with it. Alternatively, STREAMS drivers can be coded as loadable Linux drivers which depend upon CSLiS. In this way, individual STREAMS drivers can be loaded and unloaded dynamically. 

CSLiS is designed to provide Communications Server scheduling control in the kernel to provide efficent and balanced CPU utilization for SNA networking tasks.

The CSLiS package provides aggressive multi-tasking in multiple CPU SMP environments which was first provided in the LiS-2.12 package.

CSLiS is licensed using the GNU General Public Library License. This means that you can link proprietary STREAMS drivers with CSLiS and load the entirety into the Linux kernel without violating license restrictions.
